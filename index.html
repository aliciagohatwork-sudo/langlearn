Connect to Supabase

One-time setup to store your vocabulary in the cloud — synced across all your devices.

Step 1
Go to supabase.com, sign up for free, and create a new project.
Step 2
In your project, go to SQL Editor and run this query to create the table:
create table entries (
  id uuid default gen_random_uuid() primary key,
  subject text not null,
  level text not null,
  lesson text not null,
  character text not null,
  pronunciation text,
  notes text,
  created_at timestamptz default now()
);
Step 3
Go to Project Settings → API and copy your Project URL and anon public key below.
